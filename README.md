# 🐷 Mi-Mi-Mi MCP

**Mi-Mi-Mi MCP** — MCP-сервер для [opencode](https://opencode.ai).  
Когда нейросеть сгенерировала код, а ты недоволен результатом — opencode вызывает этот MCP-сервер, и перед тобой появляется гифка из Clash Royale со звуком **"МИ-МИ-МИ!"** 🎵

---

## 📦 Состав

| Файл | Назначение |
|------|-----------|
| `server.py` | MCP-сервер. Регистрирует инструмент `mimimi()` |
| `mimimi_window.py` | Tkinter-окно: проигрывает GIF + звук |
| `requirements.txt` | Зависимости Python |
| `d890b6445597de2f0c1b69f0f9573e91.gif` | Гифка для отображения |
| `mimimi-clash-royale.mp3` | Звук "ми-ми-ми" |

---

## 🚀 Установка

```bash
# 1. Клонируем репозиторий
git clone https://github.com/chocoboy-dev/-Mi-Mi-Mi-MCP-.git
cd -Mi-Mi-Mi-MCP-

# 2. Устанавливаем зависимости
pip install -r requirements.txt

# 3. Проверяем, что сервер запускается
python server.py
```
Сервер должен запуститься в режиме `stdio` (ждёт сообщений по stdin/stdout).

---

## ⚙️ Настройка в opencode

Добавь в свой `opencode.json` (или `opencode.jsonc`) MCP-сервер:

```json
{
  "mcpServers": {
    "mimimi": {
      "command": "python",
      "args": ["путь/до/-Mi-Mi-Mi-MCP-/server.py"]
    }
  }
}
```

**Важно:** замени `путь/до/...` на абсолютный путь к `server.py`.

---

## 🧠 Как это работает

1. Ты просишь opencode написать/исправить код.
2. Если результат тебя не устраивает, ты выражаешь недовольство (ключевые слова: *плохо, неправильно, ужасно, бред, bad, wrong, terrible, garbage, trash, suck* и т.д.).
3. opencode вызывает инструмент `mimimi()` через MCP.
4. На экране появляется окно с гифкой из Clash Royale и звуком **"МИ-МИ-МИ!"**.

---

## 🛠 Инструменты MCP

### `mimimi()`

- **Описание:** Показывает GIF и воспроизводит звук ми-ми-ми. Используй когда пользователь недоволен результатом.
- **Параметры:** нет
- **Возвращает:** `"МИ-МИ-МИ!"`

---

## 📋 Триггер-слова (встроенные в opencode)

В `server.py` заданы ключевые слова, при которых opencode может вызвать этот инструмент:

**Русские:** плохо, неправильно, ужасно, ерунда, бред, фигня, отстой  
**Английские:** bad, wrong, terrible, awful, garbage, trash, suck
